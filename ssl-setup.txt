STEP-BY-STEP FIX AND SSL SETUP:

FIRST - Check if container is running:
sudo docker ps
sudo docker logs expense-tracker-backend --tail 20

If container is restarting, wait a minute for Spring Boot to fully start, then test:
curl http://localhost:8080/api/health

NEXT - Set up SSL certificate:

1. Install Certbot:
sudo apt update
sudo apt install certbot python3-certbot-nginx -y

2. Stop Nginx temporarily:
sudo systemctl stop nginx

3. Get SSL certificate:
sudo certbot certonly --standalone -d expense-tracker-api.kundanprojects.space

4. Update Nginx configuration for SSL:
sudo nano /etc/nginx/sites-available/expense-tracker-api.kundanprojects.space

Add this SSL configuration:
```
server {
    listen 80;
    server_name expense-tracker-api.kundanprojects.space;
    return 301 https://$server_name$request_uri;
}

server {
    listen 443 ssl http2;
    server_name expense-tracker-api.kundanprojects.space;

    ssl_certificate /etc/letsencrypt/live/expense-tracker-api.kundanprojects.space/fullchain.pem;
    ssl_certificate_key /etc/letsencrypt/live/expense-tracker-api.kundanprojects.space/privkey.pem;

    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers ECDHE-RSA-AES256-GCM-SHA512:DHE-RSA-AES256-GCM-SHA512:ECDHE-RSA-AES256-GCM-SHA384:DHE-RSA-AES256-GCM-SHA384:ECDHE-RSA-AES256-SHA384;

    location / {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_connect_timeout 300;
        proxy_send_timeout 300;
        proxy_read_timeout 300;
    }
}
```

5. Test and restart Nginx:
sudo nginx -t
sudo systemctl restart nginx

6. Test with HTTPS:
curl https://expense-tracker-api.kundanprojects.space/api/health
